# Customer Lifetime Value (CLV) & Churn Prediction

## Overview

A machine learning project that predicts Customer Lifetime Value (CLV) and customer churn using customer, transaction, engagement, and support data.

## Objectives

* Predict customer lifetime value.
* Identify customers likely to churn.
* Generate actionable business insights.

## Technologies

Python, Pandas, NumPy, Matplotlib, Scikit-Learn, Jupyter Notebook

## Models

* Random Forest Regressor (CLV Prediction)
* Random Forest Classifier (Churn Prediction)

## Features

* Customer Information
* Transaction History
* Customer Activity
* Support Tickets
* RFM Features (Recency, Frequency, Monetary)

## Key Insights

* Higher engagement leads to higher CLV.
* Low activity increases churn risk.
* Better customer support improves retention.

## Recommendations

* Focus on high-value customers.
* Re-engage inactive users.
* Improve support quality.
* Use personalized retention campaigns.
